Jezikolovac

  • See on GitHub

Jezikolovac is a language learning game developed for Virtual Reality.

The player starts the game in a spaceship, where he can select one of the three languages available (English, Spanish and Chinese) by pressing a button. By pulling the lever, a transportation pad, and a toy gun, and the level task appear.

The task is comprised of 15 cubes that have foreign language words written, and when those cubes are picked up by the player, the corresponding word is spoken in the foreign language. The player must pick up a cube that corresponds to the required word in Croatian and put it in the tube. Each error causes the cubes to be reset.

By using the teleporter, user is transported to level one, where he needs to use the gun to shoot at objects in the scene. If the wright object is hit, a particle effect highlights the object and the foreign word for that object is pronounced. The player must find all 15 object and learn their foreign word. He can teleport back to the spaceship at any time to try and complete the task and move on to the next level.

Unity 3D, C#, VRTK
