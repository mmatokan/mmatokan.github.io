Agility

  • See on GitHub

Agility is a desktop application developed as an individual project for course Object Design.

The goal of the application is easier coordination of a team working on a project using Scrum framework. After a successful login or signup, user can see all his projects. Each project has a time remaining or finished status. A user can create a new project, define a start and an end date, and add other users as members working on project.

Once a project is selected, dashboard shows projects statistics, like time remaining, percentage completed, and number of requirements done. In the Backlog tab, requirements are created, and priority is assigned to each requirement. Requirements are comprised of tasks, and when a new sprint is created, task are selected for that sprint. Each member can select a task by dragging and dropping a task card to the IN-PROGRESS column, and otherwise track their progress on the task. Once all the tasks from a requirement have been completed, the requirement is considered done.

C#, .NET, WinForm, SQLite
